##  Overview

A flutter to-do-list app developed using Firebase, project used to elevate flutter skill by building simple project. 
<br>

##  Features

#### User
- Login
- Signup
- Add New Task
- Delete Task
- Upload Profile Picture
- Edit Task

- Basically all to-do-list app features
